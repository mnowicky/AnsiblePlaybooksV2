# Ansible Playbooksv2

## Playbooks Overview

- **Multi-Tier Application Deployment**: Deploys a web and API tier with conditional setups.
- **HAProxy Load Balancer Configuration**: Configures a load balancer with dynamic backend servers.
- **Dynamic Inventory from EC2**: Gathers EC2 instances based on tags for dynamic inventory.
- **Application Update with Rollback**: Updates an application with backup and rollback capabilities.
- **Manage Users and Groups**: Automates user and group management with SSH key provisioning.
- **Automated SSL Certificate Management**: Manages SSL certificates using Let's Encrypt.
- **CI/CD Pipeline Setup**: Configures a Jenkins pipeline for continuous integration and deployment.
- **Multi-OS Configuration Management**: Installs packages based on the operating system dynamically.
